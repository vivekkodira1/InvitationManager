What this application is

  • This is just a web page (uses HTML + JavaScript). Your content is not sent to a server or stored onto a file. So its *very* important that you save the raw content yourself
  • It attempts to replace using spreadsheets (which is how most people manage invitee lists today) to add/edit invitees

Using this application

  • The tab: 'Add/Edit Invitees' allows you to add invitees, their details (ex: addresses) and dependents
  • The "Invitees List" displays the complete list of invitees and allows you to search the list using various criteria
  • The "Summary" tab displays summary information such as the number of invitees, the number of attendees, etc.
  • The tab "Raw content" can be used to copy content generated by the system to a file or to import content
  • Click "Save As" and choose "Webpage, Complete" to save this application to your local computer to use offline

Functionality coming soon
